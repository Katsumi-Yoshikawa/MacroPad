# MacroPad

[English](README.md)

## 概述

MacroPad 是一款原生 macOS 工具，用于录制、编辑、保存、触发和回放键盘及鼠标宏。程序使用 SwiftUI 与 AppKit 开发，所有功能均在本机运行。

> 首个 GitHub 版本仅发布源码，不提供预编译、正式签名或经过 Apple 公证的应用程序包。

## 功能

- 录制鼠标按键按下/松开、滚轮、键盘按下/松开以及修饰键变化。
- 录制时有意忽略连续鼠标移动和鼠标拖动事件。
- 编辑事件延迟、点击坐标、鼠标按键、键盘按键、修饰键和滚动增量。
- 启用、禁用或删除单个事件。
- 将宏保存为带版本号的 JSON 文件，并在下次启动时自动重新打开最近使用的宏文件。
- 调整回放速度和循环次数。
- 将鼠标按键设置为三种触发模式之一：单次触发、按住循环或点击切换循环。
- 主窗口隐藏后，仍可通过 macOS 顶部菜单栏使用录制、回放和触发功能。
- 拖动分隔线调整三个工作区的宽度，并在下次启动时恢复宽度。

## 系统要求

- macOS 12 Monterey 或更高版本
- 包含 Swift 5.7 或更高版本的 Xcode Command Line Tools

MacroPad 不依赖第三方软件包。

## 从源码构建

在仓库根目录运行：

```bash
swift test
./script/build_and_run.sh
```

脚本会在 `dist/MacroPad.app` 生成本地开发版本并打开。该脚本不用于生成正式签名的发布版本。

如果只需构建 Swift 可执行文件：

```bash
swift build
```

## 系统权限

MacroPad 遵循 macOS 隐私控制，不会绕过系统权限检查。

- **输入监控（Input Monitoring）**：录制输入以及监听鼠标按键触发器时需要。
- **辅助功能（Accessibility）**：回放键盘和鼠标事件时需要。

请在系统提示时前往 **系统设置 > 隐私与安全性** 授予权限。如果新权限没有立即生效，请退出并重新打开 MacroPad。

## 使用方法

1. 授予所需的 macOS 权限。
2. 使用 **Record** 开始录制；鼠标移动和拖动事件会被忽略。
3. 使用 **Stop** 或停止录制快捷键结束录制。
4. 选择事件并编辑其延迟和支持的参数。
5. 在右侧检查器中设置宏速度、循环次数以及可选的鼠标按键触发器。
6. 使用 **Play** 运行选中的宏。
7. 使用 **Save** 和 **Load** 管理宏 JSON 文件。
8. 使用宏列表下方的垃圾桶按钮从列表中移除所选宏；对应的 JSON 文件仍会保留在磁盘上。

关闭主窗口只会隐藏窗口，不会退出 MacroPad。可通过顶部菜单栏中的 MacroPad 项目重新显示窗口、控制录制/回放、查看权限状态或退出程序。

## 全局快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Option-Command-R` | 开始或停止录制 |
| `Option-Command-P` | 开始或停止回放 |
| `Option-Command-S` | 停止录制，并从录制结果末尾移除该停止快捷键产生的事件 |

快捷键目前不可自定义。如果其他程序已经使用相同组合键，快捷键注册可能失败。

## 宏文件与隐私

宏文件是人类可读的 JSON 文件，保存在用户选择的位置。MacroPad 使用本地 macOS 书签数据记录最近打开或保存的文件。

MacroPad 不包含网络功能，也不会上传录制事件或宏文件。宏可能包含敏感按键内容或点击位置，因此在分享或提交到代码仓库前，请先检查 JSON 文件。

## 已知限制

- 仅支持 macOS；首个版本不包含预编译或经过 Apple 公证的应用。
- 不录制连续鼠标移动和鼠标拖动。
- 同一时间只能回放一个宏。
- 全局快捷键不可配置。
- 不支持图像识别、条件脚本、定时执行或指定应用/窗口。
- 部分应用可能拒绝程序生成的输入。应用或游戏的服务条款也可能限制自动化，请仅在允许自动化的场景中使用 MacroPad。
